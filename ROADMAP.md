# Roadmap

Stage Passport is actively under development. The roadmap below reflects planned product, platform, and infrastructure improvements.

---

## Backend & Platform

### High Priority

- Migrate fully to PostgreSQL as the primary production database
- Background job processing for catalog synchronization
- Event-driven architecture using Kafka or Google Pub/Sub
- Duplicate detection and automatic entity merging
- Dedicated development and staging environments
- Improved backend observability and monitoring

### Future

- Distributed caching
- Rate limiting
- API versioning
- Public developer API

---
